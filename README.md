## Installation

1. Clone repository.

```shell
git clone https://github.com/minj-shalom/pwa-app.git
```

2. Get in the project folder.

```shell
cd pwa-app
```

3. Install dependencies via npm or yarn.

```shell
npm install
```

```shell
yarn
```

## Quick start

Run development server.

```shell
npm start
```

```shell
yarn start
```
